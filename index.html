A Django-based web application designed to streamline hostel operations, including managing hostels, floors, rooms, and bed allocations. The system features a dashboard for key statistics, user registration, and occupancy analytics, ensuring efficient management of hostel resources..

Event Management System (EVM):

A Django-based web application that simplifies event management by organizing events, managing vendors, tracking volunteers, and handling ticket bookings. Key features include user authentication, a dashboard for event insights, vendor and volunteer management, ticket booking with payment integration, and SMS confirmations.
